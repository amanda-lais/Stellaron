Stellaron is a Python program capable of generating a graph for each of all 88 modern constellations.


★ Forked from a group project for the graph theory class in my uni to develop further both for fun and acquiring a better understanding about multiple concepts. That includes this currently very site! It's quite simple right now, but I hope to continuously develop it and Stellaron.


★ Constellation data used comes from Stellarium.
★ Making use of Skyfield and NetworkX.
